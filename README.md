# @haixee/prettier-config

Prettier 通用配置

## 安装

```bash
yarn add -D @haixee/prettier-config
```

如果还没有安装 `prettier`，请先安装

```bash
yarn add -D prettier
```

然后在 `package.json` 中添加

```json
{
  "prettier": "@haixee/prettier-config"
}
```
